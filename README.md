# Geospatial-data-pipeline
Script to build an End-to-End Geospatial Data Pipeline with Python. 
Geospatial analysis often starts with data from multiple sources: spreadsheets, CSV files, shapefiles, APIs, or open-data portals. Before mapping or modeling, the data must be cleaned, validated, standardized, and connected to geography.
This workflow uses a sample dataset of service locations and neighbourhood boundaries, but the same approach can be applied to healthcare access, retail site analysis, transit planning, environmental monitoring, or institutional research.
